# New PR Review Enhancements

## Quick Actions (No AI Required)

1. **Delete file** — Quick removal without confirmation dialogs
2. **Rename file** — In-place renaming
3. **Copy file path** — One-click copy to clipboard
4. **Open in editor** — Jump to file in default editor
5. **View file diff** — Show just this file's changes

## Keybinding Ideas

- **Leader + R** — Remove all comments marked as "fixed"
  - Batch removes resolved/dismissed comments without confirmation
  - Useful for cleaning up feedback after fixes are applied
